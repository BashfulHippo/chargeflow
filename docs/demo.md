# ChargeFlow Demo Flow

## Pre-Demo Setup
- Open app in browser (localhost:3000 or deployed URL)
- Ensure demo mode is ON (green banner visible)
- Have dashboard tab ready to switch to

---

## Demo Script (~3 minutes)

### Part 1: Landing Page (30 sec)
1. Start on landing page
2. Highlight the tagline: "cheapest, cleanest, grid-friendly"
3. Point out the three key stats
4. Click "Plan My Charge"

### Part 2: Consumer Charging Form (45 sec)
1. Show demo mode banner: "Demo Mode - Using sample data"
2. Walk through the form:
   - Vehicle: Tesla Model 3 (60 kWh)
   - Current charge: 20%
   - Target charge: 80%
   - Ready by: 8:00 (tomorrow morning)
   - Priority: Balanced
3. Click "Generate Smart Schedule"

### Part 3: Schedule Results (60 sec)
1. **Summary Metrics**: Point out total energy needed (36 kWh)
2. **Charging Schedule Chart**:
   - "See how charging is spread across overnight hours"
   - "These green bars show when you'll actually charge"
3. **Cost Comparison**:
   - "If you charged immediately, you'd pay €X"
   - "ChargeFlow saves you €Y by shifting to off-peak"
4. **Carbon Comparison**:
   - "We also cut your CO2 by Z grams"
5. **Detailed Table**:
   - "Every hour is optimized—look at the grid load column"
   - "We avoid charging when the grid is above 75%"

### Part 4: City Dashboard (45 sec)
1. Navigate to Dashboard
2. **Adoption Slider**:
   - Start at 1,000 drivers
   - "With just 1,000 early adopters..."
   - Slide to 20,000
   - "Now look what happens at 20,000 drivers"
3. **Impact Metrics**:
   - Peak reduction: "X MW off the evening peak"
   - CO2 avoided: "That's like planting Y trees"
4. **Grid Load Chart**:
   - "The gray line is business as usual"
   - "The green line shows ChargeFlow's impact—smoother, more balanced"

### Part 5: Wrap-up (30 sec)
1. Click on a different scenario (e.g., "Sunny Weekend")
2. "On sunny days, we push more charging to midday when solar is abundant"
3. Return to landing page
4. "That's ChargeFlow—smart charging for Lisbon."

---

## Backup Plans

### If charts don't render:
- Focus on the numeric metrics
- "The visualization is loading, but look at these numbers..."

### If demo mode breaks:
- Click "Reset Demo" button
- Or manually enter demo values

### If asked about data sources:
- "We use patterns based on Portuguese OMIE market data"
- "Carbon intensity reflects Portugal's generation mix"
- "Grid load follows typical Lisbon demand curves"
